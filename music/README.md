# Music

1. `make proxy`
1. `make browser`
1. https://music.163.com/song?id=1818353205
1. Play song
1. Kill the proxy
1. Examine requests visually `mitmweb -r dumpfile`
1. `make export`
1. Rename the main m4a file to `input.m4a`
1. Copy metadata into `meta.txt`
1. `make assemble`
